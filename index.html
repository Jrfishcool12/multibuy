Welcome to multibuy

1 · Set a password

This encrypts your wallet keys on this device. You'll enter it each time you open multibuy. It's never stored or sent anywhere — if you lose it, your wallets can't be recovered.

2 · Which chain?

You can use the other one later — this just sets up your first wallets and network.

3 · Add wallets

Generate fresh wallets, or import keys you already have. You can add more or back them up anytime from the Wallets manager.

or import existing keys

4 · Network & token

An RPC is your connection to the blockchain. The free public one rate-limits with several wallets — grab a free key from Helius (Solana) or Alchemy (EVM) for reliability.

✓ You're set up

Click Connect & load wallets to see balances. Then pick Buy, Sell, or Transfer, choose wallets, Preview, and Confirm.
⚠️Start small — try one wallet first. Automated tools (DCA, auto-sell, Jito) move real funds; the red STOP ALL halts everything.
